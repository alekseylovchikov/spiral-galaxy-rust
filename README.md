# Galaxy application

### Author: Aleksey Lovchikov

#### Technologies:
- rust
- bevy

#### How to (rust install required):
Clone repo:
```shell
git clone https://github.com/alekseylovchikov/spiral-galaxy-rust.git
```
Change directory:
```shell
cd spiral-galaxy-rust
```
Run cargo build:
```shell
cargo build
```
And then run project:
```shell
cargo run
```
